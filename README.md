# 🎮 Simon Game

A modern web-based implementation of the classic **Simon** memory game. Test your memory skills with color sequences that get progressively more challenging. Built using **HTML**, **CSS**, and **JavaScript**.

## ✨ Features

- 🔁 **Challenging Gameplay** – Sequences grow longer each round
- 🔊 **Audio-Visual Feedback** – Unique sound for each button
- 💾 **Progress Tracking** – Saves high scores using local storage
- 📱 **Responsive Design** – Optimized for both desktop and mobile devices
- 🎨 **Customizable** – Easily modify colors, sounds, and difficulty

---

## 🎮 How to Play

1. Click the **START** button
2. Watch the sequence of flashing buttons
3. Repeat the sequence by clicking the buttons in the correct order
4. Advance through 20+ increasingly difficult levels
5. Try to beat your previous high score!
